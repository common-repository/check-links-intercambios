Este plugin es lo mas básico para manejar los enlaces que se consiguen para un sitio web específico.

== Description ==

El plugin Check Links Online viene listo para su uso, se guarda en la base de datos de wordpress creando una tabla
que seguramente la tendras toda la vida en tu sitio web, donde registrarás todo lo relacionado con tus intercambios,
compras, reseñas y todo tipo de enlace conseguido para tu sitio.

== Installation ==

Es sumamente sencilla, la Versión 1.0

1. Upload la carpeta `check-links-intercambios` to the `/wp-content/plugins/` directory
2. Activar el plugin atraves del menú de 'Plugins' en WordPress

== Inconveniente ==

El plugin busca en formato normal de enlaces a un sitio, si este dentro de la etique <a href="sitio.com">enlace</a>
Si le ponen el formato de estilo dentro del <a entonces no funcionará, en todo caso queda la forma manual para la búsqueda del mismo.
